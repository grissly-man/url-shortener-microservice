url-shortner-microservice 1.0.0

  • User Story: I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • User Story: If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  • User Story: When I visit that shortened URL, it will redirect me to my original link.

API Usage

/{short_url}

Looks up a {short_url} and redirects the user to its corresponding page

For example:

https://short-url-logan.herokuapp.com/2

would redirect a user to

http://www.google.com

/new/{url}

Takes a {url} string and returns a new short_url

For example:

https://short-url-logan.herokuapp.com/new/http://www.google.com
{"original_url":"http://www.google.com","short_url":"short-url-logan.herokuapp.com/2"}
